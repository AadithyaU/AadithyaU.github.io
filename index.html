Daily price

Daily price android app was submitted as part of Abacus (an intracollege symposium of Computer Science department CEG). It provides an easy to use interface for both the seller and the buyer by providing online database for the seller to insert, delete and change price and category of the product and latest UI features for the buyer such as navigation drawer for the categories and action bar search for easy search of products. It also allows the user to add ‘to-buy notes’. The shopping cart list is sent to the buyer via email along with the contact details using which seller can make arrangements for delivery.Google's 1-T n-gram is used as corpus which has over 1 billion words.

The project was done 2 years ago.The code used wont be the most efficient one and also is without comments.Any query, u can contact me through the contact form.
